Old Games

In May of 2011 I got employed at Frozenbyte as a part-time QA tester.

This game was made on december 2011 as a school project. To complete a level you need to press W, then S etc. or A and then D etc. and also you need to use arrow keys. You need to catch the red pill, but first you need to make yourself slim to do it and to make yourself slim, you need to slam yourself to different sides of the playfield.

I decided to go for Ludum Dare 23, where you need to do game in 48 hours. I did this in early 2012.

I liked these Ludum Dares, so I did this one in April 2013. It has very low usability, but yeah I made that too! http://ludumdare.com/compo/ludum-dare-26/?action=preview&uid=8260

September 2014 I did this for Ludum Dare. I think it was alright! I actually already worked as a full-time game designer at this point at Frozenbyte. http://ludumdare.com/compo/ludum-dare-30/?action=preview&uid=8260

Then I kind of stopped doing these game jams where I used crappy Flash development tools and started learning Unity late 2015 and early 2016. I left Frozenbyte and started to prototype my own games.

2018 October I'm releasing Super X Chess on Steam.
